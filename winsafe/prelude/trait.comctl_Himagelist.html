Available on crate feature comctl only.
Expand description

This trait is enabled with the comctl feature, and provides methods for HIMAGELIST.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

fn Add(self, hbmp_image: HBITMAP, hbmp_mask: Option<HBITMAP>) -> SysResult<u32>

ImageList_Add method.

Note: A copy of the bitmap is made, and this copy is then stored. You’re still responsible for freeing the original bitmap.

Examples
use winsafe::prelude::*;
use winsafe::{co, HIMAGELIST, SIZE};

let himgl = HIMAGELIST::Create(
    SIZE::new(16, 16), co::ILC::COLOR32, 1, 1)?;

himgl.Destroy()?;

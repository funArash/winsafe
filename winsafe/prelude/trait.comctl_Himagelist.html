This trait is enabled with the comctl feature, and provides methods for HIMAGELIST.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

fn BeginDrag(
    &self,
    itrack: u32,
    hotspot: POINT
) -> SysResult<HimagelistDragGuard<'_>>

ImageList_BeginDrag method.

In the original C implementation, you must call ImageList_EndDrag as a cleanup operation.

Here, the cleanup is performed automatically, because BeginDrag returns an HimagelistDragGuard, which automatically calls ImageList_EndDrag when the guard goes out of scope. You must, however, keep the guard alive, otherwise the cleanup will be performed right away.

use winsafe::prelude::*;
use winsafe::{HIMAGELIST, POINT};

let himgl: HIMAGELIST; // initialized somewhere

let _drag = himgl.BeginDrag(0, POINT::new(0, 0))?; // keep guard alive

use winsafe::prelude::*;

let himgl = HIMAGELIST::Create(
    SIZE::new(16, 16), co::ILC::COLOR32, 1, 1)?;

himgl.Destroy()?;

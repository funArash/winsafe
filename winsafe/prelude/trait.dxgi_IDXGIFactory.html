This trait is enabled with the dxgi feature, and provides methods for IDXGIFactory.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

fn iter_adapters(&self) -> Box<dyn Iterator<Item = HrResult<IDXGIAdapter>> + '_>

Returns an iterator over the IDXGIAdapter elements which calls IDXGIFactory::EnumAdapters internally.

Examples
use winsafe::prelude::*;
use winsafe::IDXGIFactory;

let factory: IDXGIFactory; // initialized somewhere

for adapter in factory.iter_adapters() {
    let adapter = adapter?;
    // ...
}

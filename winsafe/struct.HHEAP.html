#[repr(transparent)]
pub struct HHEAP(_);
Available on crate feature kernel only.
Expand description

Handle to a heap object. Originally just a HANDLE.

If you’re allocating memory using just the default process heap – that is, by calling HHEAP::GetProcessHeap –, consider using the HeapBlock high-level abstraction.

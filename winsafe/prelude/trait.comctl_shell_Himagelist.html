This trait is enabled with comctl and shell features, and provides methods for HIMAGELIST.

Prefer importing this trait through the prelude:

use winsafe::prelude::*;

source

fn add_icon_from_shell(self, file_extensions: &[&str]) -> SysResult<()>

Calls SHGetFileInfo to retrieve one or more shell file icons, then passes them to AddIcon.

Examples
use winsafe::prelude::*;
use winsafe::{co, HIMAGELIST, SIZE};

let himgl = HIMAGELIST::Create(
    SIZE::new(16, 16), co::ILC::COLOR32, 1, 1)?;

himgl.add_icon_from_shell(&["mp3", "wav"])?;

himgl.Destroy()?;

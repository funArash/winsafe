Struct winsafe::co::LVS_EX

#[repr(transparent)]
pub struct LVS_EX(_);
Available on crate feature comctl only.
Expand description

Extended list view control styles (u32).

This is an extended windoow style, convertible to WS_EX.

This is a bitflag constant, which implements the NativeBitflag trait.

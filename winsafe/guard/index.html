Module winsafe::guard

source ·
Expand description

RAII implementation for handles.

Most handles require cleanup routines, but some handles require cleanup routines only under certain circumstances, what makes it impossible to universally implement these routines for the handle types.

So, when a handle needs a cleanup routine, it is instead returned wrapped into a guard.

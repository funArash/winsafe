IMediaSeeking COM interface over IMediaSeekingVT. Inherits from IUnknown.

Automatically calls IUnknown::Release when the object goes out of scope.

Examples

use winsafe::prelude::*;
use winsafe::{IGraphBuilder, IMediaSeeking};

let graph_builder: IGraphBuilder; // initialized somewhere

let media_seeking = graph_builder
    .QueryInterface::<IMediaSeeking>()?;

pub fn new(parent: &impl GuiParent, opts: ProgressBarOpts) -> ProgressBar

Instantiates a new ProgressBar object, to be created on the parent window with HWND::CreateWindowEx.

Panics

Panics if the parent window was already created – that is, you cannot dynamically create a ProgressBar in an event closure.

source

pub fn new_dlg( parent: &impl GuiParent, ctrl_id: u16, resize_behavior: (Horz, Vert) ) -> ProgressBar

Instantiates a new ProgressBar object, to be loaded from a dialog resource with HWND::GetDlgItem.

Panics

Panics if the parent dialog was already created – that is, you cannot dynamically create a ProgressBar in an event closure.

Struct winsafe::HeapBlock

source · 
pub struct HeapBlock { /* private fields */ }
Available on crate feature kernel only.
Expand description

Manages an HHEAP memory block which uses the default heap of the calling process – that is, calls HHEAP::GetProcessHeap.

The memory block is freed automatically when the object goes out of scope.

Examples

use winsafe::{self as w, prelude::*};

let mut bl = w::HeapBlock::alloc(10)?;

for (idx, b) in bl.as_mut_slice().iter_mut().enumerate() {
    *b = (idx * 10) as _;
}

for b in bl.as_slice().iter() {
    print!("{} ", b);
}
println!("");

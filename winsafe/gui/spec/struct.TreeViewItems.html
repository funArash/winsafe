Struct winsafe::gui::spec::TreeViewItems

source · 
pub struct TreeViewItems<'a> { /* private fields */ }
Available on crate feature gui only.
Expand description

Exposes item methods of a TreeView control.

You cannot directly instantiate this object, it is created internally by the control.
